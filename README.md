# Dog Stay（ペットサロンサイト）

## 概要
架空のペットサロンのWebサイトを想定して制作しました。  
ユーザーがサービス内容を把握しやすく、スマートフォンからでも閲覧しやすいUIを意識しています。

---

## 目的
フレームワークを使わず、HTML/CSS/JavaScriptのみで  
レイアウト設計とレスポンシブ対応の理解を深めるために制作しました。

---

## 主な実装
- レスポンシブ対応（スマートフォン表示）
- セクションごとのレイアウト設計
- ナビゲーションUI
- 画像とテキストの視認性を意識した配置

---

## 使用技術
- HTML
- CSS
- JavaScript

---

## 工夫した点
- スマートフォン表示時に読みやすくなる余白と文字サイズを調整
- ユーザーの視線の流れを意識したレイアウト構成
- シンプルな構造で読み込み速度を意識

---
